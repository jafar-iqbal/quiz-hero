Q1: What is JavaScript, and how is it used in web development?

Answer: JavaScript is a versatile, high-level programming language primarily used for creating and controlling dynamic website content. It allows developers to implement complex features such as interactive forms, animations, and real-time updates, enhancing the user experience.

Q2: What is the difference between let, var, and const in JavaScript?

Answer: `var` is function-scoped and can be redeclared, `let` is block-scoped and cannot be redeclared within the same block, and `const` is also block-scoped but is used for variables that should not be reassigned. `const` ensures immutability for primitive data types but does not make objects immutable.

Q3: What is a JavaScript promise, and how does it work?

Answer: A JavaScript promise is an object representing the eventual completion or failure of an asynchronous operation. Promises provide a way to handle asynchronous tasks by allowing methods such as `then` for handling resolved values and `catch` for handling errors, improving code readability and manageability.

Q4: How does event delegation work in JavaScript?

Answer: Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements. By leveraging the event bubbling mechanism, the parent element can handle events for dynamically added or removed child elements, reducing memory usage and improving performance.

Q5: What is the Document Object Model (DOM) in JavaScript?

Answer: The DOM is a programming interface for web documents, representing the page so that programs can change the document structure, style, and content. JavaScript can interact with the DOM to manipulate HTML elements, allowing developers to create dynamic and interactive web pages.
